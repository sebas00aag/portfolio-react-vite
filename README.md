# Portfolio React (Vite)

## Requisitos
- Node.js 18+ recomendado

## Instalar
```bash
npm install
```

## Correr en local
```bash
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

### ¿Qué incluye?
- React + Vite
- Secciones: Home / About / Projects / Contact
- i18n simple ES/EN con localStorage
- Menú responsive tipo sidebar
- Botón scroll-to-top
- Proyectos como data (`src/data/projects.js`)
